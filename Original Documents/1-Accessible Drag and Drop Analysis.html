Difficult Play with object in a 2D plane, while dragging object can mutate, when it is released object position might change Web Application Move in a 2D play area, transfer electrons from sweater to balloon, rub balloon on an insulator, release balloon and observe it respond to electrostatic forces. Keyboard/Mouse/Touch WASD and arrow keys move balloon in 2D plane. Hold down shift plus movement keys to move balloon more slowly. Difficult Example of speed impacting drag interaction Web Application Example: Drag an object across another object where speed of dragging impacts behavior. Example: A magnet dragging thorugh a coil of wire connected to a lightbulb. Change in drag speed will change induced current through the wire and brightness of the lightbulb. Keyboard/Mouse/Touch WASD and arrow keys move object in a 2D plane. Hold down shift to change movement speed. As object moves through critical locations, aria-live is used to announce changes in the application. For instance, if magnet moves quickly to the left and the lightbulb lights up, a summary of this is announced with aria-live. https://phet.colorado.edu/sims/html/friction/latest/friction_en.html https://phet.colorado.edu/sims/html/faradays-law/latest/faradays-law_en.html These examples currently has no support for accessibility.
